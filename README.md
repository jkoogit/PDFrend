# PDFrend Project Management

환영합니다! PDFrend 프로젝트는 단순한 기능 구현을 넘어, 개발 과정에서의 성장과 체계적인 의사결정을 목표로 합니다.

본 저장소는 프로젝트의 모든 기획, 설계, 기술, 의사결정 및 관리 문서를 포함하며, [Docs-as-Code](https://docs.github.com/en/repositories/managing-your-repositorys-settings-and-features/customizing-your-repository/about-documentation-for-your-repository) 전략에 따라 관리됩니다.
