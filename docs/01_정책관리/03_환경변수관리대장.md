# 03. 환경 변수 관리 대장

## 1. 개요
본 문서는 PDFrend 프로젝트의 각 배포 환경별 환경 변수(Environment Variables) 및 민감 정보를 관리하기 위한 대장입니다. 보안을 위해 **실제 값은 본 문서에 기록하지 않으며**, 변수명과 용도, 설정 여부만을 관리합니다.

## 2. 관리 정책 (Policy-SEC-001)
1. **Git 업로드 금지**: 실제 값이 포함된 `.env` 파일은 절대 Git 레포지토리에 업로드하지 않는다.
2. **마스킹 처리**: 문서화 시 실제 값은 `********` 또는 `(Secret)`으로 표기한다.
3. **주입 방식**: 
   - 로컬: `.env.local` (Git 제외)
   - 배포 환경: Cloudtype 환경 변수 설정을 통해 직접 주입
4. **변경 이력**: 환경 변수 추가/변경 시 본 문서의 이력 섹션에 기록한다.
