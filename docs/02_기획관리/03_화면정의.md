# [화면정의서] PDFrend: 상세 화면 및 인터랙션 정의 (v2.0)

본 문서는 [시스템 정책서](./01_Policy_Management.md)에 정의된 정책들을 기반으로 합니다.

## 1. 메인 대시보드 및 문서 관리

![대시보드 디자인](/home/ubuntu/PDFrend_Dashboard_UI.png)
*그림 1: PDFrend 메인 대시보드 디자인안*

대시보드는 사용자가 자신의 문서와 공유된 문서를 한눈에 파악하고 관리할 수 있는 공간입니다. 좌측 사이드바를 통해 카테고리를 탐색하고, 중앙의 카드 형태 리스트를 통해 문서의 핵심 정보(제목, 저자, 수정일, 버전)를 빠르게 확인할 수 있습니다. 상단의 'Upload' 버튼을 통해 새로운 문서를 손쉽게 추가할 수 있습니다.

### 1.1 카테고리 및 문서 목록
*   **좌측 트리**: '내 문서', '공유 문서' 루트 카테고리 고정. 하위 카테고리는 계층 구조로 자유롭게 생성/이동.
*   **문서 상세 정보**: 제목, 출판사, 저자, 역자, 발행일자, 가격, 페이지, ISBN, 구매/도서 링크 등 상세 메타데이터 관리.
*   **보기 모드**: 리스트 뷰(상세 정보 위주), 썸네일 뷰(표지 이미지 위주) 전환 버튼.

### 1.2 업로드 및 버전 관리
*   **용량 관리**: [Policy-STG-001], [Policy-STG-002], [Policy-STG-003]에 따라 사용자별 용량 할당 및 공유 문서 용량 제외 정책이 적용됩니다.

*   **업로드 팝업**: 파일 선택 시 기존 문서와 매칭 시도. [새 문서로 등록 / 기존 문서의 vX.X로 등록] 선택.
*   **버전 목록**: 문서 상세에서 버전 리스트 노출. 각 버전 옆에 '변경 이력' 버튼 배치 -> 클릭 시 팝업으로 작업 로그(누가, 언제, 무엇을) 조회.

## 2. PDF 작업 뷰어 (상세)

![뷰어 디자인](/home/ubuntu/PDFrend_Viewer_UI.png)
*그림 2: PDFrend 작업 뷰어 디자인안*

작업 뷰어는 PDFrend의 핵심 기능을 수행하는 공간입니다. 상단바에서 문서의 상태를 확인하고 보기 모드를 전환할 수 있으며, 좌측 사이드바의 5개 탭을 통해 목차, 주석, 북마크 등을 체계적으로 관리합니다. 중앙의 캔버스에서는 PDF 렌더링과 함께 플로팅 툴바를 이용한 자유로운 주석 작업이 가능합니다. 주석에는 작성자 정보가 포함되어 협업 시 유용하게 활용됩니다.

### 2.1 사이드바 (5개 탭 구성)
1.  **목차**: 계층형 리스트. 편집 모드 시 핸들로 순서 변경. 항목 클릭 시 해당 페이지 이동.
2.  **주석 목록**: 페이지별 주석 리스트. 작성자/타입 필터 및 텍스트 검색 기능 포함.
3.  **북마크**: 사용자 정의 북마크 리스트. 설정된 포맷(`{페이지}*{날짜}*{작성자}`)에 따라 자동 명명.
4.  **페이지 관리**: 썸네일 리스트. 드래그로 순서 변경, 삭제, 신규 페이지 삽입 기능.
5.  **검색 결과**: 문서 내 텍스트 검색 결과 리스트. 클릭 시 해당 위치로 이동 및 하이라이트.

### 2.2 중앙 작업 영역 및 툴바
*   **주석 도구 모음**: 형광펜, 펜, 텍스트, 주석상자, 지우개, 도형(선, 화살표, 사각형), 삽입(이미지, 서명, 도장).
*   **OCR 팝업 메뉴**: 텍스트 선택 시 [복사, 목차 추가, 구글/네이버 검색, 번역, AI 검색] 메뉴 노출.
*   **보기 옵션 (Ctrl+V)**: 주석 작성자 표시 여부, 1쪽/2쪽 보기, 0페이지 표시, 스크롤 방향(가로/세로), 다크모드 설정.

## 3. 특수 기능 화면

### 3.1 공유 및 Draft 관리
*   **공유 관리 탭**: [공유 관리(내가 준 권한)], [사용 관리(내가 받은 권한)] 탭. 공유 대상, 기간, 권한 정보 테이블 표시.
*   **Draft 머지 팝업**: 오프라인 작업 후 온라인 접속 시 노출. 좌측(서버 현재), 우측(로컬 변경점) 비교 뷰 제공. 변경 항목별 체크박스로 선택적 머지 수행.

### 3.2 개인 및 시스템 설정
*   **프로필**: 별명, 아바타 이미지, 기본 작업자 이름 설정.
*   **단축키 설정**: 뷰어 기능별 단축키(Ctrl+1, Ctrl+2, Ctrl+G 등) 리스트 및 사용자 커스텀 수정 UI.
