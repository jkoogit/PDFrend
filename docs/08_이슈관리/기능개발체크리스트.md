## 🎯 기능 개요

**목표:** User 도메인의 핵심 기능을 구현하여 사용자 관리 시스템의 기반을 마련합니다.

**범위:**
- User 엔티티 및 값 객체 정의
- 비즈니스 규칙 구현 (저장소 할당량 관리 등)
- UserDomainService 구현
- UserRepositoryImpl 구현
- 유닛 테스트 작성 (95% 이상 커버리지)

**기대 효과:**
- 사용자 정보를 안전하게 관리할 수 있는 기반 마련
- 저장소 할당량 관리 자동화
- 향후 다른 도메인에서 User 도메인 활용 가능
