# 이메일/비밀번호 로컬 인증 구현 가이드

**작성일:** 2026-05-12  
**버전:** 1.0  
**상태:** 구현 가이드

---

## 📌 개요

이 문서는 Passport.js의 LocalStrategy를 사용하여 이메일/비밀번호 기반 로컬 인증을 구현하는 방법을 설명합니다. 이 방식은 카카오, 네이버, 구글 OAuth와 함께 사용되어 사용자에게 다양한 로그인 옵션을 제공합니다.

---

## 🔐 보안 원칙

### 비밀번호 저장

**절대 금지:**
- ❌ 평문 저장
- ❌ 단순 MD5/SHA1 해싱
- ❌ 단방향 암호화

**필수:**
- ✅ bcrypt를 사용한 해싱 (솔트 포함)
- ✅ 최소 10 라운드 이상의 해싱
- ✅ 비밀번호 검증 시 시간 복잡도 일정 유지

### 세션 관리

- 세션 저장소: PostgreSQL (connect-pg-simple)
- 세션 만료: 24시간 (설정 가능)
- 보안 쿠키: HttpOnly, Secure, SameSite=Strict

## 💻 구현 코드

### 1. LocalStrategy 설정

**파일: `server/_core/passport.ts`**

```typescript
import passport from 'passport';
import { Strategy as LocalStrategy } from 'passport-local';
import bcrypt from 'bcryptjs';
import * as db from '../db';

// LocalStrategy 설정
passport.use(
  new LocalStrategy(
    {
      usernameField: 'email',
      passwordField: 'password',
    },
    async (email, password, done) => {
      try {
        // 사용자 조회
        const user = await db.getUserByEmail(email);
        
        if (!user) {
          return done(null, false, { message: '이메일 또는 비밀번호가 잘못되었습니다.' });
        }

        // 비밀번호 검증
        const isPasswordValid = await bcrypt.compare(password, user.passwordHash);
        
        if (!isPasswordValid) {
          return done(null, false, { message: '이메일 또는 비밀번호가 잘못되었습니다.' });
        }

        // 계정 활성화 확인
        if (!user.isActive) {
          return done(null, false, { message: '비활성화된 계정입니다.' });
        }

        // 마지막 로그인 시간 업데이트
        await db.updateUserLastLogin(user.id);

        return done(null, user);
      } catch (error) {
        return done(error);
      }
    }
  )
);

// 세션 직렬화
passport.serializeUser((user, done) => {
  done(null, user.id);
});

// 세션 역직렬화
passport.deserializeUser(async (id, done) => {
  try {
    const user = await db.getUserById(id);
    done(null, user);
  } catch (error) {
    done(error);
  }
});

export default passport;
```

### 2. 회원가입 API

**파일: `server/routers/auth.ts`**

```typescript
import { publicProcedure, router } from '../_core/trpc';
import { z } from 'zod';
import bcrypt from 'bcryptjs';
import * as db from '../db';

const authRouter = router({
  // 회원가입
  register: publicProcedure
    .input(
      z.object({
        email: z.string().email('유효한 이메일을 입력하세요'),
        password: z.string().min(8, '비밀번호는 최소 8자 이상이어야 합니다'),
        name: z.string().min(1, '이름을 입력하세요'),
      })
    )
    .mutation(async ({ input }) => {
      // 1. 이메일 중복 확인
      const existingUser = await db.getUserByEmail(input.email);
      if (existingUser) {
        throw new Error('이미 가입된 이메일입니다.');
      }

      // 2. 비밀번호 해싱 (10 라운드)
      const passwordHash = await bcrypt.hash(input.password, 10);

      // 3. 사용자 생성
      const user = await db.createUser({
        email: input.email,
        name: input.name,
        passwordHash,
        loginMethod: 'local',
        isActive: true,
      });

      return {
        id: user.id,
        email: user.email,
        name: user.name,
      };
    }),

  // 로그인
  login: publicProcedure
    .input(
      z.object({
        email: z.string().email(),
        password: z.string(),
      })
    )
    .mutation(async ({ input, ctx }) => {
      // Passport LocalStrategy가 처리
      // 이 엔드포인트는 Express 미들웨어에서 처리됨
      return { success: true };
    }),
});

export default authRouter;
```

### 3. Express 라우터 설정

**파일: `server/_core/routes/auth.ts`**

```typescript
import { Express, Request, Response } from 'express';
import passport from 'passport';
import bcrypt from 'bcryptjs';
import * as db from '../../db';

export function registerAuthRoutes(app: Express) {
  // 회원가입 (POST)
  app.post('/api/auth/register', async (req: Request, res: Response) => {
    try {
      const { email, password, name } = req.body;

      // 유효성 검사
      if (!email || !password || !name) {
        return res.status(400).json({ error: '필수 정보를 입력하세요' });
      }

      if (password.length < 8) {
        return res.status(400).json({ error: '비밀번호는 최소 8자 이상이어야 합니다' });
      }

      // 이메일 중복 확인
      const existingUser = await db.getUserByEmail(email);
      if (existingUser) {
        return res.status(409).json({ error: '이미 가입된 이메일입니다' });
      }

      // 비밀번호 해싱
      const passwordHash = await bcrypt.hash(password, 10);

      // 사용자 생성
      const user = await db.createUser({
        email,
        name,
        passwordHash,
        loginMethod: 'local',
        isActive: true,
      });

      // 세션 생성
      req.login(user, (err) => {
        if (err) {
          return res.status(500).json({ error: '로그인 처리 중 오류가 발생했습니다' });
        }
        res.json({ success: true, user });
      });
    } catch (error) {
      console.error('[Auth] Register failed', error);
      res.status(500).json({ error: '회원가입 처리 중 오류가 발생했습니다' });
    }
  });

  // 로그인 (POST)
  app.post(
    '/api/auth/login',
    passport.authenticate('local', {
      failureRedirect: '/api/auth/login-failed',
    }),
    (req: Request, res: Response) => {
      res.json({ success: true, user: req.user });
    }
  );

  // 로그인 실패
  app.get('/api/auth/login-failed', (req: Request, res: Response) => {
    res.status(401).json({ error: '이메일 또는 비밀번호가 잘못되었습니다' });
  });

  // 로그아웃 (POST)
  app.post('/api/auth/logout', (req: Request, res: Response) => {
    req.logout((err) => {
      if (err) {
        return res.status(500).json({ error: '로그아웃 처리 중 오류가 발생했습니다' });
      }
      res.json({ success: true });
    });
  });

  // 현재 사용자 정보 (GET)
  app.get('/api/auth/me', (req: Request, res: Response) => {
    if (!req.user) {
      return res.status(401).json({ error: '인증되지 않은 사용자입니다' });
    }
    res.json({ user: req.user });
  });
}
```

### 4. 세션 설정

**파일: `server/_core/session.ts`**

```typescript
import session from 'express-session';
import pgSession from 'connect-pg-simple';
import { Pool } from 'pg';
import { ENV } from './env';

const pgPool = new Pool({
  connectionString: ENV.databaseUrl,
});

export const sessionMiddleware = session({
  store: new (pgSession(session))({
    pool: pgPool,
    tableName: 'session',
  }),
  secret: ENV.sessionSecret,
  resave: false,
  saveUninitialized: false,
  cookie: {
    secure: ENV.nodeEnv === 'production', // HTTPS만 사용
    httpOnly: true, // JavaScript에서 접근 불가
    sameSite: 'strict', // CSRF 공격 방지
    maxAge: 24 * 60 * 60 * 1000, // 24시간
  },
});
```

### 5. Express 앱 설정

**파일: `server/_core/index.ts`**

```typescript
import express from 'express';
import passport from 'passport';
import { sessionMiddleware } from './session';
import { registerAuthRoutes } from './routes/auth';

const app = express();

// 미들웨어
app.use(express.json());
app.use(sessionMiddleware);
app.use(passport.initialize());
app.use(passport.session());

// 라우터
registerAuthRoutes(app);

export default app;
```

## 🧪 테스트 코드

**파일: `server/auth.local.test.ts`**

```typescript
import { describe, it, expect, beforeAll, afterAll } from 'vitest';
import bcrypt from 'bcryptjs';
import * as db from './db';

describe('Local Authentication', () => {
  const testUser = {
    email: 'test@example.com',
    password: 'TestPassword123!',
    name: 'Test User',
  };

  beforeAll(async () => {
    // 테스트 데이터 정리
    await db.deleteUserByEmail(testUser.email);
  });

  afterAll(async () => {
    // 테스트 데이터 정리
    await db.deleteUserByEmail(testUser.email);
  });

  it('비밀번호를 안전하게 해싱해야 함', async () => {
    const hash = await bcrypt.hash(testUser.password, 10);
    expect(hash).not.toBe(testUser.password);
    expect(hash.length).toBeGreaterThan(20);
  });

  it('올바른 비밀번호를 검증해야 함', async () => {
    const hash = await bcrypt.hash(testUser.password, 10);
    const isValid = await bcrypt.compare(testUser.password, hash);
    expect(isValid).toBe(true);
  });

  it('잘못된 비밀번호를 거부해야 함', async () => {
    const hash = await bcrypt.hash(testUser.password, 10);
    const isValid = await bcrypt.compare('WrongPassword', hash);
    expect(isValid).toBe(false);
  });

  it('사용자를 생성하고 조회할 수 있어야 함', async () => {
    const passwordHash = await bcrypt.hash(testUser.password, 10);
    const user = await db.createUser({
      email: testUser.email,
      name: testUser.name,
      passwordHash,
      loginMethod: 'local',
      isActive: true,
    });

    expect(user.email).toBe(testUser.email);
    expect(user.name).toBe(testUser.name);

    const retrievedUser = await db.getUserByEmail(testUser.email);
    expect(retrievedUser).toBeDefined();
    expect(retrievedUser?.email).toBe(testUser.email);
  });
});
```
