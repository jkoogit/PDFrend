# 04. 공통코드사전

## 목적
그룹코드/상세코드를 표준화하여 테이블 상태값과 이벤트 코드를 일관되게 관리한다.

## 그룹코드 목록
| 그룹코드 | 그룹명 | 설명 |
| :--- | :--- | :--- |
| AUTH_STATUS | 인증상태 | 인증수단/링크 상태 코드 |
| USER_STATUS | 회원상태 | 회원 계정 상태 코드 |
| AUTH_EVENT_TYPE | 인증이벤트유형 | 인증 이벤트 유형 코드 |
| AUTH_EVENT_RESULT | 인증이벤트결과 | 인증 이벤트 처리 결과 |

## 상세코드 목록
| 그룹코드 | 상세코드 | 상세코드명 | 사용처 | 비고 |
| :--- | :--- | :--- | :--- | :--- |
| AUTH_STATUS | ACTIVE | 활성 | auth_identities, user_auth_links | 기본값 |
| AUTH_STATUS | REVOKED | 해제 | user_auth_links | 링크 해제 |
| AUTH_STATUS | HISTORY | 이력 | user_auth_links | 과거 링크 보관 |
| USER_STATUS | ACTIVE | 활성 | users | 정상 사용자 |
| USER_STATUS | SUSPENDED | 정지 | users | 제재 상태 |
| USER_STATUS | WITHDRAWN | 탈퇴 | users | 탈퇴 사용자 |
| AUTH_EVENT_TYPE | SIGNUP | 가입 | auth_events | 신규 가입 |
| AUTH_EVENT_TYPE | LOGIN | 로그인 | auth_events | 로그인 시도 |
| AUTH_EVENT_TYPE | REVERIFY | 재인증 | auth_events | 인증 갱신 |
| AUTH_EVENT_TYPE | LINK | 연결 | auth_events | 인증수단 연결 |
| AUTH_EVENT_TYPE | UNLINK | 해제 | auth_events | 인증수단 해제 |
| AUTH_EVENT_RESULT | SUCCESS | 성공 | auth_events | 처리 성공 |
| AUTH_EVENT_RESULT | FAIL | 실패 | auth_events | 처리 실패 |
