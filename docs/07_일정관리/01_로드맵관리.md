# 07. 일정 관리 (Roadmap Management)

## 목적

프로젝트의 주요 마일스톤과 상위 일정을 관리하여 전체적인 진행 상황을 파악합니다. PDFrend는 '성장 중심의 단계별 개발'을 지향하므로, 각 마일스톤은 기능 구현뿐 아니라 **설계 학습, 의사결정 기록, 정책 수립**이 함께 이루어지는 단위로 구성합니다.

```text
[완료]           [진행중]             프론트엔드 뷰어       실시간/오프라인       PWA, 관리자, 최적화
                 DB 스키마(완료)       주석 CRUD             WebSocket            성능 튜닝
                 인증/CRUD(진행중)     OCR 연동              버전 관리            인프라 이관
```
